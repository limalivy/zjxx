# 字界修仙

基于浏览器的中文汉字打字游戏，以修仙世界为主题。纯 HTML + CSS + JavaScript，无依赖，打开即玩。

## 快速开始

直接用浏览器打开 `index.html` 即可，无需安装任何东西。

## 游戏模式

### ⚡ 渡劫 — 挑战模式
- 4 分钟限时挑战
- 血条未被扣光则渡劫成功，修仙等级 +1
- 等级越高字下落越快（练气期 0.5× → 渡劫期 4.0×）
- 失败不降级，可重新挑战

### 🧘 修炼 — 自由练习模式
- 不限时，自定义下落速度（0.1–5.0 倍速）
- 支持暂停（Esc / P 键或按钮）
- **智能练习系统**：打错或漏掉的字会在 10 秒后高概率再次出现，加强训练
- 连续正确输入 2 次后恢复正常概率
- 显示本局修炼时长

### 🔥 试炼 — 无尽模式
- 初始速度 0.5 倍，每 15 秒加速 0.1 倍
- 坚持越久分数越高
- 最高分自动保存

## 字集

内置三套字集，可在主菜单切换：

| 字集 | 字数 |
|------|------|
| 常用字 500 | 500 |
| 常用字 501–1000 | 500 |
| 常用字 1001–1500 | 500 |

不同字集的渡劫等级和试炼最高分相互独立。

### 新增字集

1. 将字集 JSON 文件放入 `data/` 目录
2. 在 `data/char-sets.json` 中添加文件名
3. 无需修改代码

字集文件格式：
```json
{
  "title": "字集名称",
  "words": ["字", "一", "二", ...]
}
```

## 修仙等级

| 等级 | 速度倍率 |
|------|----------|
| 练气期 | 0.5× |
| 筑基期 | 1.0× |
| 金丹期 | 1.5× |
| 元婴期 | 2.0× |
| 化神期 | 2.5× |
| 合体期 | 3.0× |
| 大乘期 | 3.5× |
| 渡劫期 | 4.0× |

等级配置位于 `levels.json`，可自由修改。

## 操作

| 按键 | 作用 |
|------|------|
| 输入汉字 | 消除屏幕上对应掉落字 |
| Esc / P | 修炼模式暂停/继续 |
| Enter | 速度选择时确认 |

## 数据存储

游戏数据保存在浏览器 localStorage 中，包括：
- 各字集的修仙等级和试炼最高分
- 字集选择偏好
- 修炼速度设置

清除浏览器数据会导致记录丢失。

## 技术

- 游戏循环：`requestAnimationFrame`，60fps
- 输入处理：IME 组合输入兼容（compositionstart / compositionend）
- 纯前端，无框架，无构建步骤
